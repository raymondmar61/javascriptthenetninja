The Document Object Model (DOM) is an application programming interface which means use the DOM when we interact with webpages: add content to the HTML document, delete content from the HTML document, change content on a HTML document. The Document itself is the HTML document web page.

Every HTML element in the document is an object. For example, head, body, ul, p are objects.

The model starts with the hierarchy top going down. HTML at top. Head and Body are below HTML. Title is below Head. ul is below Body. li is below ul. Div is below Body. p is below Body. Everthing we can change in the document is a node: elements, text within elements, HTML attributes

HTML
Head
Body
Title ul and DIV, h1, p
  li and p (below DIV)

What can we do with the DOM? Examples are change the h1 text node at the top of the page, change the background color of an element node, animate the logo node from left to right, and expand the height of an element node when you click on it.

The DOM can reach in at any node. Once we have that node, we can traverse the DOM to access other nodes.
